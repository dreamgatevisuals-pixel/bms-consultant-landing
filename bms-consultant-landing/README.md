# BMS Consultant Landing Page

High-conversion static landing page for Belfield Management Solutions Inc. (BMS) built for LinkedIn Live / LinkedIn traffic.

## What's included
- `index.html` – page layout + copy
- `styles.css` – styling
- `script.js` – basic interactions + demo lead capture (localStorage)

## Deploy on GitHub Pages
1. Push these files to your repo (root).
2. Go to **Settings → Pages**
3. Under **Build and deployment** choose:
   - Source: `Deploy from a branch`
   - Branch: `main` / `(root)`
4. Save. GitHub will provide your live URL.

## Configure scheduling
In `index.html`, replace the placeholder scheduling URL:
- Search for `https://example.com`
- Replace with your Calendly / HubSpot / GoHighLevel booking link

## Lead capture
The form currently stores leads in the browser's localStorage for demo purposes.
To collect real leads, connect the form to a provider like HubSpot, GoHighLevel, Zapier, or Netlify Forms.
